## 🌐 After Deployment

### Step 1: Add Custom Domain (5 min)
1. In Vercel → Settings → Domains
2. Add: `highendinteriors.co.in`
3. Update DNS in GoDaddy:
   - Add CNAME: `cname.vercel-dns.com`
   - Or use Vercel nameservers

### Step 2: Submit to Google (5 min)
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add property: `https://highendinteriors.co.in`
3. Verify ownership (DNS method)
4. Submit sitemap: `https://highendinteriors.co.in/sitemap.xml`

### Step 3: Submit to Bing (2 min)
1. Go to [Bing Webmaster Tools](https://www.bing.com/webmasters)
2. Add site: `https://highendinteriors.co.in`
3. Submit sitemap

### Step 4: Add Google Analytics (5 min)
1. Create account at [Google Analytics](https://analytics.google.com)
2. Get tracking ID
3. Add to `app/layout.tsx` (optional but recommended)
